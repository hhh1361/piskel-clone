Regular expressions
regexp = new RegExp("pattern", "flags");
regexp = /pattern/; // no flags
regexp = /pattern/gmi; // with flags g, m and i
let str = "I love JavaScript!"; // will search here
let regexp = /love/;
str.search(regexp) // 2
A regular expression consists of a pattern and optional flags: g, i, m, u, s, y.
Methods of RegExp and String
str.search(reg)
let str = "A drop of ink may make a million think";
str.search( /a/i ) // 0 (first match at zero position)
str.match(reg), no “g” flag
let str = "Fame is the thirst of youth";
result = str.match( /fame/i ) );
result[0] // Fame (the match)
result.index // 0 (at the zero position)
result.input // "Fame is the thirst of youth" (the string)
str.match(reg), with “g” flag
let str = "HO-Ho-ho!";;
result = str.match( /ho/ig ) );
console.log( result ); // HO, Ho, ho (array of 3 matches, case-insensitive)
str.matchAll(regexp)
str.split(regexp|substr, limit)
'12-34-56'.split('-')// array of [12, 34, 56]
'12-34-56'.split(/-/)// array of [12, 34, 56]
str.replace(str|reg, str|func)
'12-34-56'.replace("-", ":") // 12:34-56
'12-34-56'.replace( /-/g,":" ) // 12:34:56
regexp.exec(str)
let str = 'A lot about JavaScript at https://javascript.info';
let regexp = /javascript/ig;
let result;
while (result = regexp.exec(str)) {
alert( `Found ${result[0]} at ${result.index}` );
// shows: Found JavaScript at 12, then:
// shows: Found javascript at 34
}
regexp.exec(str)
let str = 'A lot about JavaScript at https://javascript.info';
let regexp = /javascript/ig;
regexp.lastIndex = 13;
let result;
while (result = regexp.exec(str)) {
alert( `Found ${result[0]} at ${result.index}` );

}

Character classes
\d (“d” is from “digit”)
\s (“s” is from “space”)
\w (“w” is from “word”)
\d\s\w
Word boundary: \b
"Hello, Java!".match(/\bJava\b/)// Java
"Hello, JavaScript!".match(/\bJava\b/)// null
Inverse classes
\D
\S
\W
\B
A dot is any character
"Z".match(/./) // Z
Escaping, special characters
[ \ ^ $ . | ? * + ( )
Sets and ranges [...]
Sets
[tm]op
"Mop top".match(/[tm]op/gi) // "Mop", "top"
Ranges
[a-z] [0-5]
"Exception 0xAF".match(/x[0-9A-F][0-9A-F]/g) // xAF
Excluding ranges
[^aeyo]
[^0-9]
[^\s]
Quantity {n}
The exact count: {5}.
"I am 12345 years old".match(/\d{3}/)// "123"
The range: {3,5}, match 3-5 times.
+ Means “one or more”, the same as {1,}.
? Means “zero or one”, the same as {0,1}.
*Means “zero or more”, the same as {0,}.
The lazy mode of quantifier
?
\d+
\d+?
Capturing groups
'Gogogo now!'.match(/(go)+/i) // "Gogogo"
Using/go+/ we can find goooo or gooooooooo, but not Gogogo
Alternation (OR) |
gr(a|e)y// gray or grey
gra|ey// gra or ey
